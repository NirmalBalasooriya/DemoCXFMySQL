Camel Project for Spring 
=========================================

To build this project use

    mvn install

Install project in to FUSE enter following command in fuse console

osgi:install -s mvn:com.mycompany/camel-spring/1.0.0-SNAPSHOT

After successful run you can be able to access following URL

    http://localhost:9292/cxf/serviceInterface?wsdl
